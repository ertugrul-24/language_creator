## üéØ Get Started

### Prerequisites
- Node.js 20.10.0+
- npm 10.2.3+
- Git

### Quick Setup (15 minutes)

```bash
# 1. Clone repository
git clone https://github.com/your-username/language_creator.git
cd language_creator
npm install

# 2. Choose your backend
# Path A: FREE (Supabase) - Perfect for learning
# Path B: PAID (Firebase) - Production ready
# See SETUP.md for detailed guides

# 3. Start development server
npm run dev
# Open http://localhost:5173
```

### Deployment Options

**Free Path (Supabase):**
- Frontend: Vercel (free)
- Backend: Supabase (free tier)
- **Cost: $0/month**
- Best for: Learning, prototyping, community projects

**Paid Path (Firebase):**
- Frontend: Vercel (free or paid)
- Backend: Firebase (pay-as-you-go)
- **Cost: $25-75+/month at scale**
- Best for: Production apps, commercial use

Both paths use identical code. Choose based on your needs.

## üèóÔ∏è Tech Stack

### Frontend
- **React 18** - Modern UI framework with hooks
- **TypeScript** - Full type safety (strict mode)
- **Tailwind CSS** - Utility-first styling with dark mode
- **React Router** - Client-side navigation
- **Vite** - Fast development server and build tool

### Backend (Choose One)

**Supabase (Free/Learning):**
- PostgreSQL database
- Real-time subscriptions with LISTEN/NOTIFY
- Built-in authentication (Email, Google, GitHub)
- Free tier: 500MB storage, unlimited API calls
- Self-hosting support via Docker

**Firebase (Production/Enterprise):**
- Firestore NoSQL database
- Firebase Authentication
- Automatic scaling and global CDN
- Estimated cost: $25-75/month at typical usage

### Deployment
- **Frontend:** Vercel (free tier available)
- **Backend:** Supabase Cloud (free) or Firebase
- **Cost:** $0/month (free tier) or $25-75+/month (production)

## ‚ú® Quick Start

### Installation
```bash
git clone https://github.com/your-username/language_creator.git
cd language_creator
npm install

# Choose backend: Supabase (free) or Firebase (paid)
# Follow SETUP.md for detailed setup
npm run dev
```

Open http://localhost:5173

### Tech Stack

**Frontend:** React 18, TypeScript, Tailwind CSS, React Router, Vite

**Backend Options:**
- **Supabase** (Free) - PostgreSQL, real-time, 500MB free storage
- **Firebase** (Paid) - Firestore, auth, $25-75/month

**Deployment:** Vercel (free) + Supabase Cloud (free) or Firebase
